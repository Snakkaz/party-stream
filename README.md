## Getting Started

Simply open `index.html` in your web browser to start using Party Stream!

### Usage

1. Enter a name for your party in the input field
2. Click "Start Stream" to begin streaming
3. Watch your viewer count grow!
4. View all active streams in the "Active Streams" section
5. Click "Stop Stream" when your party is over

## Technical Details

This is a client-side only application built with:
- HTML5
- CSS3 (with animations)
- Vanilla JavaScript (ES6+)

No server or build process required - just open and go!
